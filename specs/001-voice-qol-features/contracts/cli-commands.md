# CLI Command Contracts: Voice QoL Features

**Branch**: `001-voice-qol-features` | **Date**: 2026-03-01

## voice-agent init

One-time interactive setup wizard for configuring the voice agent system.

```
voice-agent init [--discord-token TOKEN] [--tts-backend local|elevenlabs]
                 [--tts-voice NAME] [--elevenlabs-api-key KEY]
                 [--elevenlabs-voice-id ID] [--whisper-model SIZE]
                 [--speech-mode pause|stop_token] [--stop-word WORD]
                 [--default-cli claude|codex] [--terminal EMULATOR]
                 [--server-host HOST] [--server-port PORT]
                 [--skip-mcp] [--skip-daemon] [--non-interactive]
```

### Flags

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--discord-token` | string | — | Discord bot token |
| `--tts-backend` | string | "local" | TTS backend: "local" or "elevenlabs" |
| `--tts-voice` | string | "Ryan" | Default voice profile name |
| `--elevenlabs-api-key` | string | — | ElevenLabs API key |
| `--elevenlabs-voice-id` | string | — | ElevenLabs voice ID |
| `--whisper-model` | string | "base" | Whisper model size |
| `--speech-mode` | string | "pause" | Default speech completion mode |
| `--stop-word` | string | "over" | Stop word for stop-token mode |
| `--default-cli` | string | "claude" | Default CLI for /spawn |
| `--terminal` | string | auto-detect | Terminal emulator override |
| `--server-host` | string | "127.0.0.1" | HTTP server bind address |
| `--server-port` | int | 8765 | HTTP server bind port |
| `--skip-mcp` | flag | false | Skip MCP registration in CLI tools |
| `--skip-daemon` | flag | false | Skip systemd daemon setup |
| `--non-interactive` | flag | false | Don't prompt interactively (require all flags) |

### Config File Output

Written to `~/.config/voice-agent/config.env`:

```env

ANTHROPIC_API_KEY=***
TTS_BACKEND=local
TTS_VOICE=Ryan
TTS_DEVICE=cuda

WHISPER_DEVICE=cuda
SPEECH_MODE=pause
STOP_WORD=over
DEFAULT_CLI=claude

SERVER_PORT=8765
ROUTER_ENABLED=false
```

### Entry Point

Registered as a console script in `pyproject.toml`:
```toml
[project.scripts]
voice-agent = "server.init:main"
```

### Exit Codes

| Code | Meaning |
|------|---------|
| 0 | Success |
| 1 | Configuration validation failed |
| 2 | Required flag missing in non-interactive mode |
| 130 | User cancelled (Ctrl+C) |

## voice-agent serve

Start the voice agent server (alternative to `python -m server.main`).

```
voice-agent serve [--transport stdio|http] [--host HOST] [--port PORT]
```

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--transport` | string | "http" | Transport mode: "stdio" or "http" |
| `--host` | string | from config | HTTP bind address |
| `--port` | int | from config | HTTP bind port |

### Behavior

- Reads config from `~/.config/voice-agent/config.env` (fallback to `.env` in working directory)
- If `--transport http`: starts Starlette + uvicorn HTTP server
- If `--transport stdio`: starts MCP stdio server (legacy single-client mode)
- Both modes start the Discord bot in a background thread
